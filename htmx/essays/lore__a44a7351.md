# htmx lore

Carson Gross

December 17, 2024

<img src="/img/i-lied.png" style="border-radius: 32px; max-width: 100%" alt="I lied, I don&#39;t have a Twitter account. Take off your shoes, we&#39;re going to learn about hypermedia." />

For better or <a href="https://x.com/IroncladDev/status/1866185587616596356" rel="noopener" target="_blank">for worse</a>, htmx has collected a lot of lore, mainly around <a href="https://twitter.com/htmx_org" rel="noopener" target="_blank">the twitter account</a>.

Here are some explanations.

## <a href="#it-s-so-over-we-re-so-back" class="zola-anchor" aria-label="Anchor link for: it-s-so-over-we-re-so-back">It’s So Over/We’re So Back</a>

A common set of phrases used by htmx enthusiasts when, for example, <a href="https://x.com/bunjavascript/status/1708557665268568412" rel="noopener" target="_blank">@bunjavascript told me to delete my account</a>

## <a href="#htmx-ceo" class="zola-anchor" aria-label="Anchor link for: htmx-ceo">htmx CEO</a>

At one point there was a hostile takeover attempt of the htmx CEO position and, in a desperate poison pill, I declared everyone CEO of htmx.

<a href="https://x.com/gitpush_gitpaid" rel="noopener" target="_blank">Turk</a> created <a href="https://htmx.ceo" rel="noopener" target="_blank">https://htmx.ceo</a> if you want to register as a CEO.

If someone emails hr@bigsky.software asking if you are CEO of htmx, I will tell them yes.

You can put it on your LinkedIn, because it’s true.

## <a href="#laser-eye-horse" class="zola-anchor" aria-label="Anchor link for: laser-eye-horse">Laser Eye Horse</a>

At some point I photoshopped lasers onto a horse mask, as kind of an homage to <a href="https://x.com/horse_js" rel="noopener" target="_blank">@horse_js</a>.

For some reason it stuck and now it’s the <a href="https://swag.htmx.org/products/i-lied-sticker" rel="noopener" target="_blank">official unofficial</a> mascot of htmx.

## <a href="#spieltrieb" class="zola-anchor" aria-label="Anchor link for: spieltrieb">Spieltrieb</a>

Spieltrieb means “play instinct”, and is a big part of the <a href="https://x.com/search?q=spieltrieb%20from%3Ahtmx_org&amp;src=typed_query" rel="noopener" target="_blank">htmx vibe</a>.

## <a href="#pickles" class="zola-anchor" aria-label="Anchor link for: pickles">Pickles</a>

At some point someone (I think <a href="https://x.com/techsavvytravvy" rel="noopener" target="_blank">@techsavvytravvy</a>), generated <a href="https://x.com/htmx_org/status/1708697536587047142" rel="noopener" target="_blank">a grug AI image</a>, and there was a pickle smiling in a really bizarre way in it.

So we started riffing on pickles and now <a href="https://swag.htmx.org/products/htmx-pickle-shirt" rel="noopener" target="_blank">there’s a shirt</a>.

Cry more, <a href="https://x.com/DrizzleORM/status/1757149983713665238" rel="noopener" target="_blank">drizzle</a>.

## <a href="#xss" class="zola-anchor" aria-label="Anchor link for: xss">XSS</a>

In July 2023, when htmx first got popular, there was a <a href="https://x.com/htmx_org/status/1683607693246775297" rel="noopener" target="_blank">moral panic</a> around <a href="https://x.com/htmx_org/status/1683529221195571200" rel="noopener" target="_blank">cross site scripting</a>. I <a href="https://x.com/htmx_org/status/1683607217499414531" rel="noopener" target="_blank">may</a> have <a href="https://x.com/htmx_org/status/1683649190071791617" rel="noopener" target="_blank">overcooked</a> my <a href="https://x.com/htmx_org/status/1683612179512057856" rel="noopener" target="_blank">response</a> to <a href="https://x.com/htmx_org/status/1683818711763877892" rel="noopener" target="_blank">it</a>.

## <a href="#shut-up-warren" class="zola-anchor" aria-label="Anchor link for: shut-up-warren">Shut Up Warren</a>

<a href="https://x.com/WarrenInTheBuff" rel="noopener" target="_blank">@WarrenInTheBuff</a> is the king of twitter and we regularly fight with him. This often ends in someone saying <a href="https://x.com/ThePrimeagen/status/1792564215749779515" rel="noopener" target="_blank">“shut up warren”</a>.

You can see the htmx website do this by going to <a href="https://htmx.org?suw=true" rel="noopener" target="_blank">https://htmx.org?suw=true</a>

## <a href="#microsoft-purchase-rumor" class="zola-anchor" aria-label="Anchor link for: microsoft-purchase-rumor">Microsoft Purchase Rumor</a>

In mid-January of 2024 I got really serious with the htmx twitter account and started <a href="https://x.com/htmx_org/status/1745930477825868044" rel="noopener" target="_blank">quote</a> tweeting <a href="https://x.com/htmx_org/status/1745915394626351315" rel="noopener" target="_blank">things</a> <a href="https://x.com/htmx_org" rel="noopener" target="_blank">about microsoft</a>. People started <a href="https://x.com/SusSoftware/status/1746206195461878113" rel="noopener" target="_blank">worrying</a>. I announced a <a href="https://x.com/htmx_org/status/1746736273728094323" rel="noopener" target="_blank">license change</a> to get people freaked out about a rug pull.

<a href="https://x.com/htmx_org/status/1746880860723544211" rel="noopener" target="_blank">I then changed htmx to BSD0</a>

<a href="https://x.com/htmx_org/status/1746895016256328079" rel="noopener" target="_blank">This is the offer</a> I got from microsoft (real).

## <a href="#same-thing" class="zola-anchor" aria-label="Anchor link for: same-thing">(same thing)</a>

I believe that <a href="https://x.com/htmx_org/status/1672264927136952322" rel="noopener" target="_blank">this tweet</a> is the origin of the (same thing) meme

## <a href="#stronger-together" class="zola-anchor" aria-label="Anchor link for: stronger-together">Stronger Together</a>

In December 2023, I was trying to get some indonesian twitter users to take a look at htmx, so I created a <a href="https://x.com/htmx_org/status/1734371865156563428" rel="noopener" target="_blank">“Montana &amp; Indonesia, Stronger Together!”</a> tweet w/an AI image.

This turned into a <a href="https://x.com/search?q=%22stronger%20together%22%20from%3A%40htmx_org&amp;src=typed_query&amp;f=live" rel="noopener" target="_blank">whole series of tweets</a>.

## <a href="#hinges" class="zola-anchor" aria-label="Anchor link for: hinges">Hinges</a>

Sometimes I am accused of being “unhinged” but, in fact, <a href="https://x.com/search?q=from%3Ahtmx_org%20hinges&amp;src=typed_query" rel="noopener" target="_blank">I own many hinges</a>.

## <a href="#library" class="zola-anchor" aria-label="Anchor link for: library">* library</a>

People often [call htmx a framework](https://htmx.org/essays/is-htmx-another-javascript-framework/), but it’s <a href="https://x.com/htmx_org/status/1848751101035827210" rel="noopener" target="_blank">a library</a>

## <a href="#man" class="zola-anchor" aria-label="Anchor link for: man">“man”</a>

A common <a href="https://x.com/search?q=%22man%22%20from%3Ahtmx_org&amp;src=typed_query&amp;f=live" rel="noopener" target="_blank">one word response</a> when I don’t feel like arguing with someone.

## <a href="#the-le-marquee-d-something" class="zola-anchor" aria-label="Anchor link for: the-le-marquee-d-something">The Le Marquee d’&lt;something&gt;</a>

In December 2024, I <a href="https://github.com/bigskysoftware/htmx/commit/2b88d967c19619281228d1bf5398751615bdf462" rel="noopener" target="_blank">added a marquee tag</a> to the htmx website and started using the honorific (sic) in my twitter title.

## <a href="#htmx-sucks" class="zola-anchor" aria-label="Anchor link for: htmx-sucks">htmx sucks</a>

I wrote an essay called [htmx sucks](https://htmx.org/essays/htmx-sucks/) in which I criticize htmx (some valid, some tongue in cheek, most both.) I also released <a href="https://swag.htmx.org/products/htmx-sucks-mug" rel="noopener" target="_blank">a mug</a> that I will often link to when people are criticizing htmx.

## <a href="#jason-knight" class="zola-anchor" aria-label="Anchor link for: jason-knight">Jason Knight</a>

Jason Knight <a href="https://x.com/JasonKn99664124/status/1731555036864381251" rel="noopener" target="_blank">hates htmx</a> and wrote a <a href="https://archive.is/rQrl7" rel="noopener" target="_blank">great post</a> about it.

Please don’t harass him, <a href="https://x.com/htmx_org/status/1756476449693872635" rel="noopener" target="_blank">I draw energy</a> from his posts.

## <a href="#drop-downs" class="zola-anchor" aria-label="Anchor link for: drop-downs">Drop Downs</a>

In July 2023, sparked by the accusation that htmx users could not create dropdowns, I did a deep-dive into web drop down technology and <a href="https://x.com/htmx_org/status/1684936514885869568" rel="noopener" target="_blank">uncovered a bombshell</a>

## <a href="#htmx-is-a-front-end-library-of-peace" class="zola-anchor" aria-label="Anchor link for: htmx-is-a-front-end-library-of-peace">“htmx is a front end library of peace”</a>

A phrase I will often <a href="https://x.com/search?q=htmx%20is%20a%20front%20end%20library%20of%20peace%20from%3A%40htmx_org&amp;src=typed_query&amp;f=live" rel="noopener" target="_blank">quote tweet</a> violent htmx-related imagery with.

## <a href="#the-process-tm" class="zola-anchor" aria-label="Anchor link for: the-process-tm">The Process ™</a>

<a href="https://x.com/htmx_org/status/1697651918858764375" rel="noopener" target="_blank">The Process™</a> is the mechanism by which people initially hostile to htmx come to be enlightened.

## <a href="#that-s-ridiculous" class="zola-anchor" aria-label="Anchor link for: that-s-ridiculous">“that’s ridiculous”</a>

In <a href="https://x.com/htmx_org/status/1807183339222405317" rel="noopener" target="_blank">June 2023</a>, <a href="https://twitter.com/srasash" rel="noopener" target="_blank">@srasash</a> accused htmx of being a government op, the first in many such increasingly ridiculous claims. I typically quote-tweet these claims and point out that <a href="https://x.com/search?q=%22that%27s%20ridiculous%22%20from%3A%40htmx_org&amp;src=typed_query&amp;f=live" rel="noopener" target="_blank">“that’s ridiculous”</a>

## <a href="#grug" class="zola-anchor" aria-label="Anchor link for: grug">Grug</a>

I created <a href="http://grugbrain.dev" rel="noopener" target="_blank">http://grugbrain.dev</a>.

## <a href="#the-htmx-intercooler-js-feud" class="zola-anchor" aria-label="Anchor link for: the-htmx-intercooler-js-feud">The htmx/intercooler.js feud</a>

The htmx & <a href="https://x.com/intercoolerjs" rel="noopener" target="_blank">intercooler.js</a> twitter accounts often fight with one another. Sometimes its just me <a href="https://x.com/intercoolerjs/status/1859652045399355559" rel="noopener" target="_blank">switching back and forth</a>, but two other people have access to the intercooler account, so sometimes I have no idea who I am fighting with.

## <a href="#if-nothing-magically-works" class="zola-anchor" aria-label="Anchor link for: if-nothing-magically-works">If Nothing Magically Works</a>

Nothing <a href="https://x.com/htmx_org/status/1729870461864226829" rel="noopener" target="_blank">magically breaks</a>.

## <a href="#r-webdev" class="zola-anchor" aria-label="Anchor link for: r-webdev">/r/webdev</a>

I was very unfairly given <a href="https://x.com/htmx_org/status/1719687461385691283" rel="noopener" target="_blank">a lifetime ban</a> from <a href="https://www.reddit.com/r/webdev/" rel="noopener" target="_blank">/r/webdev/</a> for an <a href="https://old.reddit.com/r/webdev/comments/17i0loi/anyone_heard_of_htmx/" rel="noopener" target="_blank">obviously satirical post</a>. Even the term “htmx” is banned (or semi-banned) on that sub, so people now use the <a href="https://swag.htmx.org/products/htmeggs-shirt" rel="noopener" target="_blank">htmeggs</a> instead.

## <a href="#looking-into-this" class="zola-anchor" aria-label="Anchor link for: looking-into-this">“looking into this”</a>

<a href="https://x.com/search?q=%22idk%22%20from%3Ahtmx_org&amp;src=typed_query&amp;f=live" rel="noopener" target="_blank">idk</a>, I just think <a href="https://x.com/search?q=%22looking%20into%20this%22%20from%3Ahtmx_org&amp;src=typed_query" rel="noopener" target="_blank">it’s funny</a>

## <a href="#look-at-this-nerd-ragin" class="zola-anchor" aria-label="Anchor link for: look-at-this-nerd-ragin">“Look at this nerd ragin’”</a>

A common phrase used to <a href="https://x.com/search?q=%22Look%20at%20this%20nerd%20ragin%27%22%20from%3Ahtmx_org&amp;src=typed_query" rel="noopener" target="_blank">mock people (including ourselves)</a> with.

## <a href="#joker-bane-skeletor-thanos-etc" class="zola-anchor" aria-label="Anchor link for: joker-bane-skeletor-thanos-etc">Joker/Bane/Skeletor/Thanos, etc.</a>

htmx is a <a href="https://x.com/htmx_org/status/1651698199478796292" rel="noopener" target="_blank">villain</a> in the front-end world, I’m good w/that
